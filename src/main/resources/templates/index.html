This is Articulate!

The definite purpose of this application is to articulate some basic concepts and capabilities of the EKS platform, specifically the Elastic Runtime which is responsible for running application workloads.

Solution Architecture on AWS - Update on June 4

Application Architecture

articulate is a web application that exposes friendly, browsable user interface. However, it does not work with data directly. It depends on the attendee-service application to manage data. The attendee-service persists data to a MySQL database.
